    <h2 className="text-3xl font-bold text-gray-900 dark:text-white">Quick start</h2>
    <p className="mt-3 text-base text-gray-600 dark:text-gray-400">Get SupaSync running in your vault in minutes</p>
  </div>

  <Steps>
    <Step title="Install the plugin">
      Build the plugin and copy it to your vault's plugins folder:
      
      ```bash
      npm run build
      cp main.js manifest.json ~/.obsidian/plugins/supasync/
      ```
      
      Enable **SupaSync** in Obsidian Settings → Community plugins.
    </Step>

    <Step title="Set up Supabase">
      Create a Supabase project and run the database migration to create the `notes` table with Row Level Security:
      
      ```sql
      create table notes (
        id uuid primary key default gen_random_uuid(),
        user_id uuid not null,
        path text not null,
        title text,
        content text not null,
        hash text not null,
        updated_at timestamptz default now(),
        unique (user_id, path)
      );
      
      alter table notes enable row level security;
      ```
      
      See the [Supabase setup guide](/setup/supabase) for complete RLS policies and authentication configuration.
    </Step>

    <Step title="Configure and authenticate">
      Open **Settings → SupaSync** and enter your:
      - Supabase Project URL
      - Anon key
      
      Then **sign in** using email/password or Google OAuth. Authentication is required for sync.
    </Step>

    <Step title="Start syncing">
      Trigger a sync via:
      - Command palette → **Full Sync**
      - Sync panel (ribbon icon) → **Sync** button
      - Enable **Auto-sync** in settings for automatic syncing on file changes
      
      <Note>
        The status bar shows sync progress and displays the last sync time. Configure sync folders in settings to limit which folders are synced (empty = entire vault).
      </Note>
    </Step>
  </Steps>
</div>
